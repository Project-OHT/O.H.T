---------------------
Instalation :
- __WINDOWS__
1. Install File [Here!](https://github.com/Project-OHT/O.H.T/archive/refs/heads/main.zip)
2. Install Packages
```
pip install -r requirements.txt
```
- __LINUX__
1. Install File
```
git clone https://github.com/Project-OHT/O.H.T.git
```
2. Install Packages
```
cd O.H.T
pip install -r requirements.txt
```
